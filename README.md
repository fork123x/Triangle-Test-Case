Triangle-Test-Case
==================

Triangle Test Case

程序流程
1.对三条边最大值，最小值输入框的输入进行检查：
	1）检查输入是否合法（特殊符号，输入错误，是否为整数类型）
	2）检查输入是否为正数
	3）检查最大值是否大于最小值

2.根据设置的最大值及最小值产生三条边的数据
包括每条边的：
略低于最小值、最小值、略高于最小值、正常值、略低于最大值、最大值、略高于最大值

3.根据下拉列表框的选项选择测试类型进行测试
包括边界值分析（4 * 3 + 1个用例）、健壮性测试（6 * 3 + 1）个用例、最坏情况测试（5 * 5 * 5个用例）、健壮最坏情况测试（7 * 7 * 7个用例）

4.对用例进行判断，判断是否为等边三角形，等腰三角形，非等边三角形，非三角形
